## 📋 About This Project

This is my personal portfolio website showcasing my data analytics projects and skills. Built as part of my MBA in Data Analytics journey at Mizoram University, this site serves as a central hub for recruiters and collaborators to learn about my work.

### ✨ Features

- **Clean, Professional Design** - Modern and responsive layout
- **Project Showcase** - Featured data analytics projects with live demos
- **Skills Display** - Visual representation of technical competencies
- **Contact Information** - Easy ways to reach me
- **Mobile Responsive** - Works perfectly on all devices
